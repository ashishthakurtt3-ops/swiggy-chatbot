# Swiggy Annual Report RAG Assistant

A Retrieval-Augmented Generation (RAG) application to answer questions from the Swiggy Annual Report without hallucinations.

### 🔗 Source Link
The Swiggy Annual Report used in this project was sourced from:
[Swiggy Annual Report FY 2023-24](https://www.swiggy.com/corporate/wp-content/uploads/2024/10/Annual-Report-FY-2023-24-1.pdf)

### 🛠️ Tech Stack
- **Framework:** LangChain
- **LLM:** Groq (Llama-3-8b)
- **Vector DB:** FAISS
- **Embeddings:** HuggingFace (Local)
- **UI:** Streamlit
